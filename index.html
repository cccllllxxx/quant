量化交易的优势

Advantages of Quantitative Trading
Traditional

● 传统交易

  • !主观决策依赖交易员个人经验、直觉与情绪判断，易受贪婪与恐惧驱动。
  • !信息处理有限人脑难以同时跟踪数百只标的、多维度指标与海量新闻资讯。
  • !执行速度慢从信号产生到下单执行存在数秒至数分钟延迟，错失转瞬即逝的机会。
  • !回测困难策略效果难以用历史数据系统验证，优化依赖“盘感”与事后复盘。
  • !纪律性弱计划止损常被“再等等”推翻，仓位管理靠感觉而非规则。
VS
Quantitative

◆ 量化交易

  • ✓客观规则驱动策略以数学模型编码，消除情绪干扰，信号触发完全基于预设规则。
  • ✓海量数据处理可同时扫描全市场数千标的，融合量价、基本面、另类数据多维信号。
  • ✓毫秒级执行算法直连交易所，下单延迟压缩至毫秒，捕捉微小定价偏差。
  • ✓历史回测验证策略在数年历史数据上模拟运行，用夏普比率、最大回撤等指标量化评估。
  • ✓严格风控纪律仓位、止损、资金管理由程序刚性执行，拒绝为人逾越规则。

基本概念

Core Concepts
01 / Candlestick

K线

Candlestick Chart

K线（蜡烛图）以矩形实体与上下影线记录每个时间周期的开盘价、收盘价、最高价和最低价，是最基础的价格可视化工具。实体颜色反映多空力量对比。

  • 实体：开盘价与收盘价之间的矩形
  • 上影线：最高价与实体上沿的连线
  • 下影线：最低价与实体下沿的连线
  • 阳线（涨）：收盘高于开盘，中国市场为红色
  • 阴线（跌）：收盘低于开盘，中国市场为绿色
02 / Fundamental

基本面

Fundamental Analysis

基本面分析通过研究公司的财务报表、行业地位、盈利能力与成长性，评估股票的内在价值。核心逻辑是：价格终将回归价值。

  • 财务指标：营收、净利润、ROE、毛利率
  • 估值指标：PE（市盈率）、PB（市净率）、PS
  • 行业分析：竞争格局、市场份额、增长空间
  • 宏观环境：利率、政策、经济周期
  • 公司治理：管理层质量、分红政策
03 / Technical

技术面

Technical Analysis

技术面分析通过研究历史价格与成交量数据，利用图表形态和技术指标预测未来走势。核心假设：历史会重演，价格包容一切。

  • 趋势指标：MA（均线）、MACD、ADX
  • 震荡指标：RSI、KDJ、CCI
  • 成交量：OBV、量价关系分析
  • 形态识别：头肩顶、双底、三角形突破
  • 支撑阻力：关键价位与突破信号
