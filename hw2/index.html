PART I · 讲解板块

老王的五大神器

老王用卖西瓜的直觉，发明了五个分析工具。 专业术语会用金色标注，鼠标移上去可以看到解释。

第一章

西瓜价格的困惑

老王在菜市场卖西瓜。他每天最头疼的事就是定价。

周一5.0元一斤，周二4.8元（下雨了），周三5.2元，周四6.0元（突然暴热），周五5.5元。

有个老顾客问他："老王，最近西瓜到底是涨了还是跌了？"

老王张了张嘴，愣是答不上来。说涨吧，今天比昨天便宜；说跌吧，前天还只有5.2元。价格上蹿下跳，就像股票的心电图，看得人心慌。

老王意识到：单日的价格全是噪音，真正有用的是趋势。

第二章

老王的第一个发明 — 移动平均线（MA）

老王想了个笨办法：我不看单天价格，我看最近5天的平均价。

第一周平均价 = (5.0 + 4.8 + 5.2 + 6.0 + 5.5) ÷ 5 = 5.3元

第二周平均价 = 算出来是 5.8元

老王把这两个数字连成一条线，发现线在往上走！他恍然大悟：虽然每天价格乱跳，但整体在慢慢变贵。

5日均线（MA5）：反应快，看短期。
30日均线（MA30）：反应慢，看中期。
250日均线（年线）：看长期牛熊。

老王的第一个工具诞生了：移动平均线，把杂乱的价格磨平，看清大方向。

📐 老王的记账本
MA(n) = (C₁ + C₂ + ... + Cₙ) ÷ n
MA5 = (C1 + C2 + C3 + C4 + C5) / 5
其中 C 为每日收盘价，n 为计算周期。
第三章

老王的第二个发明 — MACD（两条均线的距离）

老王用了均线后，发现一个问题：均线太迟钝了。 等它拐头向上，西瓜价格已经涨了两块钱。

他琢磨："我能不能提前知道均线要拐头？"

于是他盯着 12日EMA（快线） 和 26日EMA（慢线），突然灵光一闪：

我算算这两条线差了多少！

涨价初期：12日EMA刚超过26日EMA，差值为 +0.2元。
涨价高潮：12日EMA远远甩开26日EMA，差值为 +2.0元。
涨不动了：12日EMA开始走平，26日EMA慢慢跟上，差值缩小到 +0.5元。

老王把这个差值画成一条线，叫 "白线"（DIF），代表上涨的加速度。

但白线太灵敏了，老王又对它做了个 9日平均，得到 "黄线"（DEA）。

白线上穿黄线 → 黄金交叉，进货信号。
白线下穿黄线 → 死亡交叉，清仓信号。

红柱/绿柱 = (DIF - DEA) × 2，柱状图反映多空力量对比的强弱。

📐 老王的记账本
MACD 计算公式：
DIF = EMA(12) - EMA(26)
DEA = EMA(DIF, 9)
MACD柱 = (DIF - DEA) × 2
EMA = 指数移动平均，越近的数据权重越大。
第四章

老王的第三个发明 — RSI（多空打架谁赢了）

老王又发现：光看加速度还不够，他想知道现在是买方力气大，还是卖方力气大。

于是他盯着每天的涨跌：如果今天涨了，上涨幅度记为正；如果今天跌了，下跌幅度记为负。

然后他算：最近14天里，上涨的总幅度占（上涨+下跌）总幅度的比例。

RSI = 100 × 上涨幅度 ÷ (上涨幅度 + 下跌幅度)

RSI > 70 → 买方力气太大，西瓜超买了，可能要回调。
RSI < 30 → 卖方力气太大，西瓜超卖了，可能要反弹。
RSI在50附近 → 多空平衡。

📐 老王的记账本
RSI 计算公式：
RS = 平均上涨幅度 / 平均下跌幅度
RSI = 100 - 100 / (1 + RS)
平均涨跌幅度使用 Wilder 平滑法计算，周期默认14日。
第五章

老王的第四个发明 — 布林带（给西瓜画条走廊）

老王又琢磨：价格总是上蹿下跳，但大多数时候是不是在一个"合理范围"里活动？

他修了一条"走廊"，把西瓜价格框在里面：

中轨：20日均线（平均价）。
上轨：中轨 + 2倍的"波动率"（天花板）。
下轨：中轨 - 2倍的"波动率"（地板）。

波动率越大，走廊越宽；波动率越小，走廊越窄。

老王发现了两条神奇规律：

1. 走廊突然收窄 → 市场死一般寂静，多空都在憋大招 → 即将变盘。
2. 价格贴着上轨往上走 → 多头极强，趋势强劲。
3. 价格贴着下轨往下走 → 空头极强，趋势疲软。

📐 老王的记账本
布林带计算公式：
中轨 = SMA(close, 20)
上轨 = 中轨 + 2 × StdDev(close, 20)
下轨 = 中轨 - 2 × StdDev(close, 20)
带宽 = (上轨 - 下轨) / 中轨
第六章 · 扩展了解

老王的第五个发明 — ATR（颠簸测量仪）

有了这四个工具，老王还是吃过一次大亏。

有一天他看准了行情，满仓买入。结果那天市场剧烈震荡，价格大起大落，老王在最低点被"震"了出去（止损出局），卖完股价立刻暴涨。

老王拍断大腿："不是我看错了方向，是路太颠了，我被颠下车了！"

于是他发明了 ATR（平均真实波幅）——就是测量市场"颠簸度"的尺子。

ATR = 2元 → 每天正常波动2块钱。
止损线 = 入场价 ± 2 × ATR → 放4块钱的止损，扛过正常颠簸。

📐 老王的记账本
ATR 计算公式：
TR = max(H-L, |H-Cprev|, |L-Cprev|)
ATR = SMA(TR, 14)
TR = 真实波幅，H=最高价，L=最低价，Cprev=前日收盘价。
第七章 · 批判性思考

指标会"失灵" — 老王的血泪教训

老王刚开始得意忘形，以为集齐了五个工具就能天下无敌。结果被市场狠狠教训了三次。

教训一：MACD钝化（沸水里的假死叉）

大牛市中，MACD红柱拉到极限后突然缩短，白线掉头向下，形成死叉。老王吓得赶紧全卖了。结果卖完第二天，股价继续暴涨。

在极端上涨行情里，MACD会"钝化"，卖出信号基本失效。这时候要忽略指标，持股不动。

教训二：RSI钝化（疯牛里的超买）

RSI涨到了82（远超70的超买线），老王卖出。结果RSI一路飙到90多，股价继续疯涨。

在极端行情里，RSI会一直"钝化"，超买后还能再超买。这时候不能看RSI，要看趋势。

教训三：布林带陷阱（单边行情里撑开走廊）

股价沿着布林带上轨"贴地飞行"。老王以为碰到天花板该卖了，结果股价继续突破上轨，一飞冲天。

在单边大涨里，布林带的上轨不是"天花板"，而是"风向标"——价格沿着上轨走，说明趋势极强，要持有而不是卖出。

第八章 · 综合应用

老王的终极心法 — 组合拳

经过无数次亏钱，老王终于总结出铁血五步法：

步骤工具老王的操作
第一步：看天气大盘指数 + 60日均线大盘在60日线上面 → 晴天，可以玩
第二步：定方向MA + MACD均线多头排列 + MACD 零轴上方 → 只做多
第三步：找位置布林带 + RSI价格在上轨且RSI>70 → 警惕超买
第四步：设止损ATR止损价 = 买入价 - 2×ATR
第五步：控仓位ATR波动大时轻仓，波动小时重仓
PART I · 下篇

老王的第六个发明 — OBV（能量潮）

价格可以骗人，但成交量骗不了人。

OBV 第一章

老王被骗了

自从老王集齐了均线、MACD、RSI、布林带、ATR五大神器，他在菜市场里混得风生水起，大家都叫他"西瓜王"。

但有一天，他翻车了。

那天，老王盯上了一只叫"暴风科技"的股票。MACD金叉、RSI刚过50（不超买）、布林带向上开口——三个指标同时发出买入信号，完美！

老王满仓杀入。

结果买进去之后，股价不但没涨，反而掉头向下，直接跌破止损线。老王割肉出局，亏了两个月卖西瓜的利润。

老王想不通："所有指标都告诉我该买，为什么会亏？"

他坐在西瓜堆里，突然发现问题出在哪儿了：他忘了看成交量！

他去翻那几天的交易记录，发现股价上涨的那几天，成交量一天比一天低——涨是涨了，但根本没人跟，是主力自己拉高出货，俗称无量上涨。

老王一拍大腿："我光听喇叭响（指标），没看有多少人真的在掏钱进场（成交量）！"

老王的觉醒：价格可以骗人，但成交量骗不了人。
OBV 第二章

OBV 的发明 — 能量潮记录本

老王决定：必须把成交量纳入分析系统。

但成交量天天在变，今天100万手，明天200万手，后天80万手，怎么看？他又用了那个土办法：算累计。

他拿出一本新账本，取名 "能量潮记录本"，定下规矩：

今天收盘价 > 昨天收盘价 → OBV = 前一日OBV + 今日成交量（资金流入）
今天收盘价 < 昨天收盘价 → OBV = 前一日OBV - 今日成交量（资金流出）
今天收盘价 = 昨天收盘价 → OBV不变（观望）

第一天，OBV从0开始。

第一天：涨，成交量100万 → OBV = 0 + 100 = 100万
第二天：跌，成交量80万 → OBV = 100 - 80 = 20万
第三天：涨，成交量200万 → OBV = 20 + 200 = 220万
第四天：跌，成交量50万 → OBV = 220 - 50 = 170万

老王把每天的OBV值连成一条线，发现这条线就像潮水一样起起落落。价格涨、OBV也涨 → "涨潮"；价格跌、OBV也跌 → "退潮"。

📐 老王的记账本
OBV 计算公式：
if close > close_prev: OBV = OBV_prev + volume
if close < close_prev: OBV = OBV_prev - volume
if close == close_prev: OBV = OBV_prev
OBV 第三章

OBV 的价值在哪里？

老王把OBV跟之前的五个工具做了个对比，想明白了一件事：

工具看什么数据来源
MA / MACD / RSI / 布林带价格收盘价、最高价、最低价
ATR波动幅度价格
OBV资金流向成交量

所有指标都是用价格算的，只有OBV是用成交量算的。

价格是投票的结果，成交量是投票的人数。

老王恍然大悟：之前他所有工具都在分析"投票结果"，但从来没看过"投票人数"是多少。如果投票结果涨了，但投票人数只有三个人，那这个涨是虚的。但如果投票结果涨了，投票人数有一万人，那才是真正的民心所向。

OBV就是专门看"投票人数"的。

OBV 第四章 · 核心亮点

量价背离 — 老王的核心发现

老王最引以为傲的发现，就是"背离"——这是OBV最值钱的用法。

场景一：顶背离（逃命信号）

暴风科技从10元涨到20元，但OBV不但没涨，反而在下降。老王在20元那天，看到价格还在创新高，OBV却连续三天下降。他果断清仓。三天后，暴风科技从20元直接跌到15元。老王躲过一劫。

价格创新高，OBV不创新高 → 这是主力在"边拉边出货"，赶紧跑！

场景二：底背离（抄底信号）

三一重工从30元跌到20元，所有人都在恐慌割肉。但老王打开OBV一看：价格在跌，OBV却在悄悄创新高。他在20元果断买入。一个月后，三一重工反弹到25元。

价格创新低，OBV不创新低 → 这是主力在"悄悄吸筹"，跟着买！
OBV 第五章

五步法升级为六步法

从此，老王的"铁血五步法"升级成了"终极六步法"：

步骤工具老王的操作
第一步：看天气大盘指数 + 60日均线晴天还是雨天？
第二步：定方向MA + MACD做多还是做空？
第三步：找位置布林带 + RSI超买还是超卖？
第四步：查资金OBV量价配合还是背离？
第五步：设止损ATR放多宽的止损？
第六步：控仓位ATR押重注还是轻仓？
OBV和价格方向一致 → 放心操作
OBV和价格方向背离 → 谨慎操作，甚至可以反向操作
OBV 第六章 · 终极领悟

西瓜拍打器

有一天，老王站在他的西瓜摊前，看着满地的西瓜（他觉得每颗西瓜都像一只股票），说了一番话：

以前我只盯着瓜的皮（价格），以为皮变红了瓜就甜了。后来我才发现，得敲开看看有没有人买（成交量）。没人买的红皮瓜，切开全是生的。
OBV就是我的"西瓜拍打器"——价格涨了，OBV也在涨，说明这瓜真甜，放心吃。价格涨了，OBV在跌，说明这瓜被注水了，赶紧放下。

PART III · 创新思考

参数敏感性实验

老王发现，尺子的刻度不同，量的结果也不一样——那到底用哪个刻度？拖动滑块，自己感受参数变化对指标的影响。

RSI 参数对比
7天（灵敏）vs 14天（经典）vs 21天（迟缓）—— 拖动滑块实时调整
14

观察发现：

• 周期越短（如7天），RSI波动越剧烈，信号越多但假信号也多——容易"一惊一乍"

• 周期越长（如21天），RSI越平滑，信号少但更可靠——反应慢，可能错过机会

• 没有"最好"的参数，只有最适合当前西瓜的参数。 波动大的股票适合长周期，波动小的适合短周期。

布林带参数对比
1.5σ（窄走廊）vs 2σ（经典）vs 2.5σ（宽走廊）—— 拖动滑块实时调整
2.0

观察发现：

• σ越小（如1.5），走廊越窄，突破信号频繁但可靠性低——"天花板"太矮

• σ越大（如2.5），走廊越宽，突破信号少但可靠性高——"天花板"太高，可能等不到

• 参数选择本质上是在"灵敏度"和"可靠性"之间做权衡。 这就是为什么老王说"没有最好的参数"。
