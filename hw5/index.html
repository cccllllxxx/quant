PART I · 算法讲解

三类基础分类算法

分类算法直接输出离散的交易信号（涨/跌、买/卖），与策略执行天然契合。下面逐一拆解其原理与适用边界。

算法一

逻辑回归 Logistic Regression

一种广义线性分类模型。先对特征做线性组合 z = wᵀx + b，再通过 Sigmoid 函数把结果压缩到 (0,1)：

s(z) = 1 / (1 + e^(-z)) → 输出「属于正类」的概率

当概率 ≥ 0.5 判为正类，否则为负类。可解释性强、训练快、输出具有概率意义。

优势系数直接给出特征方向；对线性可分问题高效稳定。
局限只能刻画特征与标签间的线性边界，对非线性关系表达力有限。
算法二

决策树 Decision Tree

通过递归选择最优特征与切分点，把样本空间划分成若干矩形区域，每个区域对应一个类别预测。常用指标：

信息增益(ID3) · 信息增益率(C4.5) · 基尼指数(CART)

直观易懂，能自动捕捉特征间的非线性交互，但单棵树容易过拟合、泛化较弱。

优势无需特征缩放；规则可读；天然处理非线性。
局限对训练数据敏感，方差大，易过拟合。
算法三

随机森林 Random Forest

一种集成学习方法。通过自助采样（Bagging）构建多棵决策树，每棵树在随机特征子集上训练，最终以投票/平均概率输出：

ŷ = mode{ T₁(x), T₂(x), … , Tₙ(x) }

显著降低单棵树的方差，提升稳定性与精度，是表格类数据的强力基线模型。

优势抗过拟合；精度高；可输出特征重要性。
局限模型庞大、解释性弱于单棵树；训练成本更高。
PART II · 评价指标

如何衡量模型好坏

分类模型的「准不准」不能只看准确率，尤其在正负样本不平衡或代价不对称时。下面三个指标是核心。

Metric 01

混淆矩阵

把预测与真实标签交叉，得到 TP / FP / TN / FN 四个格子，是准确率、精确率、召回率、F1 的计算基础。

Metric 02

ROC 曲线

以假正率 FPR 为横轴、真正率 TPR 为纵轴，随分类阈值变化绘制的曲线；越靠左上角，区分能力越强。

Metric 03

AUC

ROC 曲线下面积，取值 [0,1]。越接近 1 越好；=0.5 相当于随机猜测；直观理解为「正例得分高于负例的概率」。

PART III · 实证分析

双数据集实战

两个数据集、三类模型，80/20 分层抽样划分训练/测试集，统一用 scikit-learn 完成。

数据集一 · 乳腺癌（标准示例）
sklearn 内置 load_breast_cancer：569 样本 / 30 医学特征，标签 0=恶性、1=良性（0/1 二分类）。

特征与标签关系明确，是检验分类流程与模型上限的标准教材数据。

数据集二 · 股票次日涨跌（量化场景）
读取贵州茅台(600519.SH)日行情，由技术指标（收益率/动量/波动率/成交量变化/均线偏离）构造特征，标签=次日收益率是否 >0。

825 条样本。金融噪声大，是检验模型在真实市场中区分力的压力测试。

交互式 ROC 曲线
切换数据集，观察三类模型的 ROC 曲线与 AUC。曲线越贴左上角越好。
混淆矩阵（各数据集最优模型）
绿格为正确预测（TP/TN），红格为误判（FP/FN）。正类：乳腺癌=良性，股票=上涨。
AUC 对比表
两类数据集上三类模型的 AUC。≥0.8 绿、0.5–0.8 琥珀、<0.5 红。
PART IV · 结论

结论与启示

本报告系统梳理了逻辑回归、决策树、随机森林三类分类算法的原理，并介绍了混淆矩阵、ROC 曲线与 AUC等评价指标。实证以乳腺癌数据集与茅台次日涨跌数据集为例，展示了完整的机器学习建模流程：数据加载 → 训练/测试划分 → 模型训练 → AUC 计算 → ROC 曲线绘制。

实验表明：在特征—标签关系明确的医学数据上，三类模型均取得极高 AUC（逻辑回归最优 0.9954），随机森林与逻辑回归明显优于决策树；而在金融市场中，仅凭日行情技术指标的次日涨跌预测 AUC 仅接近 0.5（决策树最优 0.5635），印证了市场噪声与有效市场假说。

这提示在量化交易中应用机器学习时，需要引入更丰富的特征（基本面、另类数据、文本情绪等）与更严谨的验证框架（样本外、walk-forward、成本与滑点），方能提升模型的实盘价值。
